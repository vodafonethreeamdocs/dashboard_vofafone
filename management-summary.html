üìã Project Overview

The VodafoneThree Automated Testing Dashboard is a modern, cloud-native platform that enables any authorised person to trigger automated test flows via a simple, secure web interface. The system leverages email-based communication with Amdocs Ginger to execute complex test scenarios and return results automatically.

This solution eliminates manual test triggering, provides complete audit trails, and enables non-technical stakeholders to initiate testing workflows without direct system access. All user activities are logged for compliance and monitoring.

User selects environment, business flow, and optional customer ID, then clicks "Send Email". Action is logged with all details.
