The VodafoneThree Automated Testing Dashboard is a modern, cloud-native platform that enables any authorised person to trigger automated test flows via a simple, secure web interface. The system leverages email-based communication with Amdocs Ginger to execute complex test scenarios and return results automatically.

This solution eliminates manual test triggering, provides audit trails, and enables non-technical stakeholders to initiate testing workflows without direct system access.

1

User Sends Test Request

User logs into the dashboard, selects environment, business flow, and optional customer ID, then clicks "Send Email".

2

Email Sent to Amdocs Ginger

A structured email with subject line SITE | ENV | FLOW | CustomerID is sent to Ginger's monitored inbox.

3

Ginger Executes the Flow

Amdocs Ginger parses the email subject, identifies the test flow, and automatically executes the corresponding test automation script.

4

Results Returned via Email

Once execution completes, Ginger sends the test results and logs back to the user's email address for review.
